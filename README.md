# Paradigm (alpha)

System design interview prep — spaced-repetition cards in 5-minute sessions.

**App:** https://mayankpahadia1993.github.io/paradigm/

Found a wrong card or a bug? Use the ⚑ button inside the app — it opens a
prefilled feedback email.

This repo holds the deployed build; it is generated by `scripts/deploy.sh` in the
(private) source repo. Don't edit files here directly.
